这是一个基于 **VitePress** 构建的技术博客/文档项目，名为 "北川Coding"。

### 1. 项目概况
- **类型**：静态文档站点 / 技术博客
- **核心技术栈**：
  - **框架**：VitePress v1.6.4 (基于 Vue 3 + Vite 的静态站点生成器)
  - **语言**：TypeScript (用于配置), Markdown (用于内容)
  - **包管理**：npm (依赖定义在 `package.json`)
- **部署配置**：包含 `vercel.json`，预配置了 Vercel 部署支持。

### 2. 项目结构
```

├── docs/                   # 文档根目录
│   ├── .vitepress/         # VitePress 配置目录
```

### 3. 内容规划 (基于 `config.ts`)
项目致力于构建一个完整的 **Java 后端开发学习路径**，内容涵盖：
- **Java 语言基础**：核心语法、OOP、集合、多线程、新特性
- **数据库与持久层**：MySQL、MyBatis、Redis、NoSQL
- **Web 开发基础**：HTTP、Servlet、前端基础
- **主流框架**：Spring Framework, Spring Boot, Spring Cloud
- **中间件与高并发**：消息队列 (RabbitMQ/Kafka)、搜索引擎 (ES)、分布式技术
- **工程化与运维**：Git、构建工具、Docker、CI/CD
- **计算机基础**：数据结构、操作系统、计算机网络
- **项目实战**：博客系统、电商秒杀、微服务商城

### 4. 当前状态

### 5. 常用命令
- `npm run dev`: 启动本地开发服务器
- `npm run build`: 构建静态站点
- `npm run preview`: 预览构建后的站点
