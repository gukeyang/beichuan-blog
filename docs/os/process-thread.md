# 进程与线程

进程和线程是操作系统中的核心概念，理解它们对于掌握操作系统原理至关重要。

## 进程（Process）

进程是程序的一次执行过程，是系统进行资源分配和调度的基本单位。

### 进程的特点

- 独立性：每个进程都有独立的地址空间
- 动态性：进程是程序的一次执行过程
- 并发性：多个进程可以并发执行

## 线程（Thread）

线程是进程内的执行单元，是 CPU 调度的基本单位。

### 线程的特点

- 共享性：同一进程内的线程共享进程的资源
- 轻量级：线程的创建和切换开销小于进程
- 协作性：线程之间可以方便地共享数据

## 进程 vs 线程

| 特性 | 进程 | 线程 |
|------|------|------|
| 资源分配 | 独立地址空间 | 共享进程资源 |
| 切换开销 | 大 | 小 |
| 通信方式 | IPC | 共享内存 |
| 稳定性 | 一个进程崩溃不影响其他进程 | 一个线程崩溃可能导致整个进程崩溃 |

## 总结

进程和线程各有优缺点，在实际应用中需要根据具体场景选择合适的模型。
