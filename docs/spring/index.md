# 四、主流框架（Spring 全家桶）

Spring 是 Java 后端开发的事实标准，掌握 Spring 全家桶是进入企业的必备技能。

## 📚 学习内容

### 1. Spring Framework

- **IoC（控制反转）与 DI（依赖注入）**
- **AOP（面向切面编程）**：动态代理（JDK/CGLIB）
- **事务管理**：@Transactional 原理

### 2. Spring Boot（企业标配）

- **自动配置原理**：@EnableAutoConfiguration
- **Starter 依赖管理**
- **配置文件**：application.yml
- **整合**：MyBatis、Redis、Swagger

### 3. Spring Cloud（微服务）

- **服务注册与发现**：Nacos / Eureka
- **远程调用**：OpenFeign
- **网关**：Spring Cloud Gateway
- **限流熔断**：Sentinel / Hystrix
- **配置中心**：Nacos Config / Apollo

## 📖 推荐资源

- **书籍**：《Spring 实战》《Spring Boot 实战》
- **官方文档**：[Spring 官方文档](https://spring.io/docs)

- [ ] 理解 Spring IoC 和 AOP 原理
- [ ] 熟练使用 Spring Boot
- [ ] 掌握 Spring Cloud 微服务架构
- [ ] 能够搭建微服务项目
