## 项目介绍

**北川Coding** 是一份致力于帮助 Java 开发者构建完整知识体系的开源文档。无论你是准备校招的大学生，还是寻求进阶的职场人，这里都有你需要的核心知识。

本项目内容涵盖：

- **Java 语言基础**：深入理解 Java 核心机制
- **主流框架技术**：Spring Boot、Spring Cloud、MyBatis 等
- **中间件与架构**：消息队列、分布式缓存、高并发系统设计
- **计算机基础理论**：操作系统、网络协议、数据结构
