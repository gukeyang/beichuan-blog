# 八、项目实战

理论结合实践，通过项目实战巩固所学知识，提升工程能力。

## 📚 项目类型

### 1. 博客系统

**技术栈**：Spring Boot + MyBatis + MySQL

**亮点**：
- JWT 鉴权
- Markdown 支持
- 文章管理
- 评论系统

### 2. 电商秒杀

**技术栈**：Redis + RabbitMQ + Sentinel

**亮点**：
- 超卖控制
- 限流降级
- 高并发处理
- 分布式锁

### 3. 微服务商城

**技术栈**：Spring Cloud + Nacos + Gateway

**亮点**：
- 服务拆分
- 分布式事务
- 服务治理
- 配置中心

## 🎯 项目建议

1. **从简单到复杂**：先做博客系统，再做秒杀，最后做微服务
2. **注重代码质量**：遵循编码规范，写好注释
3. **部署上线**：将项目部署到服务器，体验完整流程
4. **持续优化**：根据性能问题不断优化

## 💡 项目亮点挖掘

- **技术选型理由**：为什么选择这个技术？
- **难点攻克**：遇到了什么问题，如何解决？
- **性能优化**：如何优化系统性能？
- **架构设计**：系统架构是如何设计的？
