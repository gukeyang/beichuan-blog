# HTTP vs HTTPS

HTTP 和 HTTPS 是两种常见的网络协议，本文将详细对比它们的区别。

## HTTP 简介

HTTP（HyperText Transfer Protocol）是超文本传输协议，是互联网上应用最广泛的网络协议。

## HTTPS 简介

HTTPS（HyperText Transfer Protocol Secure）是 HTTP 的安全版本，通过 SSL/TLS 加密传输数据。

## 主要区别

1. **安全性**：HTTPS 使用加密传输，HTTP 是明文传输
2. **端口**：HTTP 使用 80 端口，HTTPS 使用 443 端口
3. **证书**：HTTPS 需要 SSL 证书
4. **性能**：HTTPS 由于加密解密，性能略低于 HTTP

## 何时使用 HTTPS？

- 涉及敏感信息的网站（如登录、支付）
- 需要保护用户隐私的场景
- 提升网站 SEO 排名
