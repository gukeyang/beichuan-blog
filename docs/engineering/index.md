# 六、工程化与运维

工程化能力是区分初级和高级开发的重要指标，掌握这些技能能让你在工作中更高效。

## 📚 学习内容

### 1. 开发工具链

- **Git**：分支管理、PR 流程
- **Maven/Gradle**：依赖管理、多模块构建
- **Docker**：镜像构建、容器运行
- **Jenkins / GitLab CI**：自动化部署

### 2. 监控与可观测性

- **日志**：Logback + ELK
- **链路追踪**：SkyWalking / Jaeger
- **指标监控**：Prometheus + Grafana

### 3. Linux 基础

- **常用命令**：ps/top/netstat/vi
- **Shell 脚本编写**
- **项目部署**：nohup / systemctl

## 🎯 学习目标

- [ ] 熟练使用 Git 进行版本控制
- [ ] 掌握 Docker 容器化部署
- [ ] 了解 CI/CD 流程
- [ ] 能够搭建监控系统
- [ ] 掌握 Linux 常用命令
