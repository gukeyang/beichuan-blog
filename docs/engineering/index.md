# 工程化

系统设计、代码规范、DevOps、性能优化等工程实践。

## 🔥 系统设计高频考点

### 高可用设计
- 冗余设计（主从、集群）
- 故障转移（Failover）
- 限流降级
- 熔断机制

### 高并发设计
- 缓存策略（多级缓存、缓存预热）
- 异步处理（消息队列）
- 分库分表
- 读写分离

### 分布式系统
- CAP 理论和 BASE 理论
- 分布式事务（2PC、TCC、Saga、本地消息表）
- 分布式锁
- 分布式 ID 生成（雪花算法）

### 微服务架构
- 服务拆分原则
- 服务治理（注册发现、配置中心）
- API 网关
- 服务监控（链路追踪、日志聚合）

## 📝 设计模式分类

- **创建型**: 单例、工厂、建造者
- **结构型**: 代理、适配器、装饰器
- **行为型**: 观察者、策略、模板方法
