# 三、Web 开发基础

Web 开发基础是理解后端工作原理的关键，掌握这些知识有助于更好地使用框架。

## 📚 学习内容

### 1. HTTP 协议

- **请求/响应结构**
- **状态码**：200/404/500/401
- **RESTful API 设计规范**：资源化、动词语义化

### 2. Servlet & Tomcat（理解原理即可）

- **Servlet 生命周期**
- **Filter、Listener**
- **Web 容器部署流程**

### 3. 前端基础（了解即可）

- **HTML/CSS/JavaScript**：能看懂前端调用接口即可
- **Axios/Fetch**：调用后端 API

## 🎯 学习目标

- [ ] 理解 HTTP 协议原理
- [ ] 掌握 RESTful API 设计
- [ ] 了解 Servlet 生命周期
- [ ] 能看懂前后端交互
