## 技术栈

- **模型**：GPT-4、Claude、Qwen、GLM、Llama
- **框架**：LangChain、LlamaIndex、Semantic Kernel
- **向量库**：Milvus、Chroma、Pinecone、pgvector
- **部署**：Ollama、vLLM、TGI、TensorRT-LLM

## 学习路线

```
入门 → 大模型基础 → Prompt Engineering
        ↓
进阶 → RAG 实战 → Agent 开发
        ↓
高级 → 本地模型部署 → 性能优化
```

## 快速开始

如果你是第一次接触大模型开发，建议按以下顺序学习：

1. **大模型基础概念** - 了解 LLM 的基本原理和核心概念
2. **Prompt Engineering** - 掌握与大模型交互的技巧
3. **RAG 实战** - 学习如何构建检索增强生成系统
4. **Agent 开发** - 探索智能体开发的进阶内容
5. **本地模型部署** - 在本地运行大模型，保护隐私
