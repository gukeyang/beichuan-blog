# 二、数据库与持久层

数据库是后端开发的核心，掌握数据库技术是成为优秀后端工程师的必备技能。

## 📚 学习内容

### 1. 关系型数据库（MySQL 为主）

- **SQL 基础**：CRUD、JOIN、子查询、事务（ACID）
- **索引原理**：B+ 树、最左前缀、覆盖索引
- **事务隔离级别**：读未提交 → 可串行化
- **锁机制**：行锁、间隙锁、死锁排查
- **性能优化**：慢查询日志、EXPLAIN 执行计划

### 2. ORM 框架

- **MyBatis**：XML 映射、动态 SQL、一级/二级缓存
- **JPA/Hibernate**：实体映射、懒加载、缓存机制（可选）

### 3. NoSQL（加分项）

- **Redis**：5 种数据类型、持久化（RDB/AOF）、缓存穿透/雪崩/击穿解决方案、分布式锁（Redisson）
- **MongoDB**：文档模型、聚合管道（适合非结构化数据）

## 📖 推荐资源

- **书籍**：《高性能 MySQL》《Redis 设计与实现》
- **实践**：本地搭建 MySQL、Redis 环境

- [ ] 了解 MongoDB 基础
