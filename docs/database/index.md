# 数据库学习指南

数据库是后端开发的核心组件，主要分为关系型数据库 (RDBMS) 和非关系型数据库 (NoSQL)。

## 1. 关系型数据库 (MySQL)
- **核心原理**：索引 (B+树)、事务 (ACID)、锁机制 (MVCC)。
- **实战优化**：SQL 调优、分库分表、主从复制。

## 2. NoSQL (Redis)
- **核心场景**：缓存、分布式锁、排行榜、计数器。
- **高可用**：哨兵模式 (Sentinel)、集群模式 (Cluster)。

## 3. 搜索引擎 (Elasticsearch)
- **核心原理**：倒排索引。
- **场景**：全文检索、日志分析 (ELK)。
