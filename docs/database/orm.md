# ORM 框架

ORM (Object-Relational Mapping) 即对象关系映射，用于将对象模型与关系数据库模型进行转换。

## 1. MyBatis

MyBatis 是一款优秀的持久层框架，支持定制化 SQL、存储过程以及高级映射。

- **优点**：SQL 灵活，性能可控。
- **缺点**：需要编写大量 SQL，移植性差。

## 2. JPA / Hibernate

- **JPA**: Java Persistence API，Java 持久层规范。
- **Hibernate**: JPA 的一种实现，全自动 ORM 框架。

## 3. MyBatis-Plus

在 MyBatis 的基础上只做增强不做改变，简化开发，提高效率。
