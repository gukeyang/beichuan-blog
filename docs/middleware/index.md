## 为什么需要中间件

1. **解耦** - 降低系统间依赖
2. **异步** - 提升响应速度
3. **削峰** - 缓冲突发流量
4. **可靠传输** - 保证消息不丢失

## 学习建议

- 先理解核心概念
- 搭建本地环境实践
- 阅读官方文档
- 生产环境注意监控和调优
