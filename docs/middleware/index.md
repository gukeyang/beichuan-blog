# 五、中间件与高并发

中间件是构建高并发、高可用系统的关键，掌握中间件技术是后端进阶的必经之路。

## 📚 学习内容

### 1. 消息队列

- **RabbitMQ**：Exchange 类型、ACK 机制
- **Kafka**：分区、副本、高吞吐设计
- **应用场景**：异步解耦、流量削峰、日志收集

### 2. 搜索引擎

- **Elasticsearch**：倒排索引、DSL 查询、聚合分析
- **ELK 日志系统**：与 Logstash + Kibana 构成

### 3. 分布式技术

- **分布式锁**：Redis/ZooKeeper 实现
- **分布式事务**：Seata（AT/TCC 模式）、RocketMQ 事务消息
- **CAP 定理、BASE 理论**

## 🎯 学习目标

- [ ] 掌握消息队列的使用场景
- [ ] 理解 Elasticsearch 搜索原理
- [ ] 掌握分布式锁和分布式事务
- [ ] 理解 CAP 和 BASE 理论
