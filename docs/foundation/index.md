# 七、计算机基础（贯穿全程）

计算机基础是程序员的底层能力，这些知识贯穿整个学习过程，需要持续学习和巩固。

## 📚 学习内容

### 1. 数据结构与算法

- **LeetCode Hot 100**：数组、链表、二叉树、DP
- **常见算法**：排序、查找、动态规划
- **时间复杂度、空间复杂度分析**

### 2. 操作系统

- **进程/线程**：进程与线程的区别
- **内存管理**：虚拟内存、页面置换
- **I/O 模型**：阻塞、非阻塞、IO 多路复用

### 3. 计算机网络

- **TCP/IP**：三次握手、四次挥手
- **HTTP/HTTPS**：协议原理、状态码
- **DNS**：域名解析原理

## 📖 推荐资源

- **刷题平台**：LeetCode、牛客网
- **书籍**：《算法导论》《计算机网络》《操作系统概念》

## 🎯 学习目标

- [ ] 能够解决 LeetCode 中等难度题目
- [ ] 理解操作系统核心概念
- [ ] 掌握网络协议原理
- [ ] 持续刷题保持手感
