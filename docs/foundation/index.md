# 计算机基础

操作系统、计算机网络、数据结构与算法等基础知识。

## 🔥 面试高频考点

### 操作系统
- 进程 vs 线程的区别
- 进程间通信方式（IPC）
- 死锁条件和解决方案
- 虚拟内存和分页机制
- Linux 常用命令

### 计算机网络
- OSI 七层模型和 TCP/IP 四层模型
- TCP 三次握手和四次挥手
- TCP 可靠传输机制（滑动窗口、拥塞控制）
- HTTP 1.1 vs HTTP/2 vs HTTP/3
- HTTPS 加密原理
- DNS 解析过程

### 数据结构
- 数组 vs 链表
- 栈和队列的应用
- 二叉树遍历（前中后序、层序）
- 红黑树特性
- 哈希表冲突解决
- 图的最短路径（Dijkstra、Floyd）

### 算法
- 十大排序算法（时间复杂度、稳定性）
- 二分查找及变体
- 动态规划经典问题（背包、LCS、LIS）
- 滑动窗口
- 双指针
