## 为什么重要

1. **面试必考** - 大厂面试 80% 考察基础
2. **技术选型** - 理解底层才能做好架构
3. **问题排查** - 深入理解才能定位疑难问题
4. **长期发展** - 基础决定技术高度

## 学习路线

```
数据结构与算法 → 操作系统 → 计算机网络 → 编译原理
```
