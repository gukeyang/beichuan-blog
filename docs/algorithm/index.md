## 学习建议

1. **理解原理** - 不要死记硬背，理解算法背后的思想
2. **手写实现** - 至少手写一遍核心算法
3. **刷题实践** - LeetCode 对应分类练习
4. **总结模板** - 形成自己的解题模板
