# 什么是 OpenClaw

OpenClaw 是一个开源的 AI 助手框架，让你能够构建和运行个性化的 AI 助手。

## 🎯 核心理念

OpenClaw 的设计目标是让 AI 助手更加**本地化**、**可定制**和**隐私友好**。

### 与传统 AI 助手的区别

| 特性 | 传统 AI | OpenClaw |
|------|--------|----------|
| 数据存储 | 云端 | 本地 |
| 可定制性 | 有限 | 完全可控 |
| 平台连接 | 单一 | 多平台 |
| 技能扩展 | 封闭 | 开放生态 |

## 📦 核心组件

### Gateway

Gateway 是 OpenClaw 的核心服务，负责：
- 管理所有连接的平台
- 处理消息路由
- 运行 AI 助手
- 加载和执行技能

### Workspace

工作区是你助手的"家"，包含：
- **SOUL.md** - 助手的人格定义
- **USER.md** - 用户偏好和上下文
- **MEMORY.md** - 长期记忆
- **skills/** - 自定义技能
- **memory/** - 日常记忆日志

### Skills

技能是 OpenClaw 的扩展机制，让助手能够：
- 调用外部 API
- 操作文件和数据
- 与第三方服务集成
- 执行自动化任务

## 🌟 使用场景

1. **个人助手** - 管理日程、提醒、笔记
2. **开发助手** - 代码审查、GitHub 管理、部署
3. **客服机器人** - 自动回复常见问题
4. **数据分析** - 自动收集和处理数据
5. **智能家居** - 控制 IoT 设备

## 🔐 隐私与安全

OpenClaw 坚持**本地优先**原则：
- 所有数据默认存储在本地
- 外部 API 调用需要明确授权
- 敏感操作需要确认
- 支持完全离线运行
