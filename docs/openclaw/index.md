# OpenClaw 教程

OpenClaw 是一个强大的 AI 助手框架，让你能够构建个性化的 AI 助手，并连接到各种通讯平台。

## 🚀 快速开始

```bash
# 安装 OpenClaw
npm install -g openclaw

# 初始化工作区
openclaw init

# 启动助手
openclaw start
```

## 💡 核心特性

- **多平台支持** - 连接 WhatsApp、Telegram、Discord 等
- **技能系统** - 通过技能扩展助手能力
- **本地优先** - 数据保存在本地，保护隐私
- **高度可定制** - 定义你的助手人格和行为
